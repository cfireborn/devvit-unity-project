1. Edgegap Account & App Setup

  1. Go to edgegap.com → create a free account
  2. Once logged in, go to Applications → Create Application
    - App Name: compersion (must match what goes in the connector code)
    - Version Name: latest
  3. Under the version, configure Ports:
    - Add port: Name game-udp, Port 7770, Protocol UDP
    - Add port: Name game-ws, Port 7771, Protocol TCP
  4. Go to API Tokens (top-right profile menu or Settings) → Create Token → copy
   it — you'll paste it into Unity

  ---
  2. Install Edgegap Unity Plugin

  In Unity: Window → Package Manager → + → Add package by name
  com.edgegap.unity-plugin
  If that doesn't work, go to edgegap.com/unity-plugin and download the
  .unitypackage, then drag it into Unity.

  After import: Edgegap → Settings in the top menu → paste your API token.

  ---
  3. Docker Setup

  1. Install Docker Desktop from docker.com (needed to build the server image)
  2. Create a Docker Hub account at hub.docker.com — free tier is fine
  3. In Docker Desktop, sign in to Docker Hub
  4. Note your Docker Hub username — you'll use it when pushing the image

  ---
  4. Unity Inspector: Switch to Multipass

  In the Unity scene hierarchy, select the NetworkManager GameObject, then:

  1. The Tugboat component should already be there — verify its port is 7770
  2. Add Component → FishNet → Transports → Bayou — set port to 7771, leave
  protocol as TCP/WebSocket
  3. Add Component → FishNet → Transports → Multipass
  4. On the TransportManager component (also on NetworkManager): find the
  Transport field — change it from Tugboat to Multipass
  5. On the Multipass component: in the Transports list, add two entries — slot
  0 = Tugboat, slot 1 = Bayou
  6. Save the scene

  ---
  5. Create the EdgegapSettings Asset

  Once I've written the EdgegapSettings ScriptableObject (in a moment), you'll
  need to:

  1. Right-click in Project window → Create → Compersion → Edgegap Settings
  2. Paste your Edgegap API token into the Api Key field
  3. Set App Name = compersion, Version Name = latest, Port Name = game-ws
  4. Assign this asset in the EdgegapConnector Inspector field on the
  NetworkManager GameObjec
